# lizheng.me

Personal homepage built with [vinext](https://github.com/nickreese/vinext) (Vite + Next.js API).

## Features

- **Particle Animation** - Interactive particles forming around the name
- **Dark/Light Mode** - Auto-detects system preference, user toggleable
- **i18n** - English/Chinese with auto-detection
- **SEO Optimized** - Full metadata, Open Graph, Twitter Cards, sitemap, robots.txt
- **Performance** - Vite-powered, optimized images, minimal bundle

## Tech Stack

- vinext (Vite + Next.js)
- React 19
- TypeScript
- Tailwind CSS v4
- tsparticles
- next-intl
- lucide-react

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```
